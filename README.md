# softsync-product-demo

This project is an interactive product demo built as part of the SoftSync frontend recruitment test case. The goal was to showcase the platform in action through meaningful animation and simulated workflow interactions rather than static UI mockups.

---

## Design Approach

The demo closely follows SoftSync’s existing visual identity while elevating interaction polish inspired by Attio’s motion design system. Background tones, tab highlights, dotted grid accents, and subtle shadow layering were used to maintain brand consistency while adding depth and hierarchy. Each tab models real product behavior through purposeful micro-interactions, including simulated AI typing, functional CRM row selection, dynamic filtering states, conditional pipeline risk highlighting, and animated workflow notifications. An automated tab rotation system creates a guided, hands-free product walkthrough that reinforces continuity across CRM stages.

---

## Tech Stack

- React
- Custom CSS animations
- State-driven component architecture

---

## Running the Project Locally

1. Clone the repository:
   git clone https://github.com/BunniBooXx/softsync-product-demo.git
   cd softsync-product-demo

2. Install dependencies:
   npm install

3. Start the development server:
   npm start

The application will run locally at:
http://localhost:3000

## Key Interaction Features

- Simulated AI typewriter responses
- Interactive CRM table with functional row and bulk selection
- Dynamic search filtering with automated typing demonstration
- Conditional pipeline risk highlighting with animated state transitions
- Email composition flow with send feedback animation
- Automated tab sequencing for guided product storytelling
